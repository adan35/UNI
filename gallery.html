What is Github Pages?

Static Websites for you and your projects, hosted directly from your GitHub repository. Just edit, push, and your changes are live.

Host a Website?

Follow the steps:

  • First create the repository containing index.html in the root directly.
  • Goto the setting of the repository, find Github Pages option and click on it.
  • Select the master branch and save it.
  • After some time, you will be provided with the link of your live website, you can click on it.
  • It may take some time to upload the website.
